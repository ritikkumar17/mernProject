# 📋 Task Manager App

A Full Stack MERN Task Manager Application where users can:

- Signup & Login
- Create Projects
- Add Tasks
- Update Task Status
- Delete Tasks
- Manage Multiple Projects

---

# 🚀 Features

## 🔐 Authentication
- User Signup
- User Login
- JWT Token Authentication
- Protected Dashboard

## 📁 Project Management
- Create New Projects
- View All Projects
- Delete Projects

## ✅ Task Management
- Add Tasks
- Update Task Status
- Delete Tasks
- Mark Tasks as:
  - Todo
  - In Progress
  - Done

---

# 🛠️ Tech Stack

## Frontend
- React.js
- React Router DOM
- Tailwind CSS

## Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
